# 🎧 MoodTunes

MoodTunes is a web application that plays music based on the user's mood.
It helps users quickly find songs that match how they feel, making music listening more enjoyable and personalized.

---

## ✨ Features

* 🎵 Select music based on mood
* 😀 Different moods like Happy, Sad, Relaxed, and Energetic
* 📱 Simple and user-friendly interface
* ⚡ Fast and responsive design

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* React
* Vite
* Node.js
* Express.js
* YouTube APIs

---

## 🚀 How to Run the Project

1. Clone the repository

```bash
git clone https://github.com/Suprith-05/MoodTunes.git
```

2. Go to the project folders and install dependencies

For Backend:
```bash
cd MoodTunes/backend
npm install
node server.js
```

For Frontend:
```bash
cd MoodTunes/frontend/frontend
npm install
npm run dev
```
